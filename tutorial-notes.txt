
write a docker-compose.yml file that allows a nodejs and mysql docker container to communicate

this tutorial covers:
  - dockerfiles
  - docker-compose.yml
  - a bit of nodeJS

this tutorial does NOT cover:
  - nodejs basics
  - sql syntax basics
  - docker basics (things like docker run, pull, exec)

process:
  - make a github repo
  - clone in all the handlebars files and nodejs stuff
  - confirm directory layout
  - make sql dockerfile && test dockerfile 
  - make nodejs container
  - test files are communicating together correctly
  - create aws subnet (limiting traffic to port 22 & 80)
  - create aws ec2 instance. host nodejs app on port 80
